## ✨ Overview

Evalify is a full-stack MERN application that simplifies academic evaluation through role-based dashboards, AI-assisted grading workflows, analytics, and performance tracking.

The platform supports multiple user roles with dedicated features for Teachers, Students, Parents, and Administrators.
