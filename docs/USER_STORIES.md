# User Stories

## Authentication

As a user,
I want to securely log in,
So that I can access my dashboard.

---

## Teacher

As a teacher,
I want to create assignments,
So that students can submit their work.

As a teacher,
I want AI to suggest grades,
So that I can save time while maintaining control.

---

## Student

As a student,
I want to submit assignments online,
So that I don't need physical submissions.

As a student,
I want AI feedback,
So that I understand how to improve.

---

## Parent

As a parent,
I want to view my child's progress,
So that I can monitor academic performance.

---

## Admin

As an administrator,
I want to manage all users,
So that the institution remains organized.
